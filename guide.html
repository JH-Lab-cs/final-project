홈 과제 집중 가이드

프로젝트 가이드

과제와 집중 시간을 한 곳에서 관리하세요.

사용 순서

사용 방법

  1. 과제 등록: 과목명, 마감일, 중요도, 예상 시간을 입력합니다.
  2. 집중 타이머 실행: 공부할 때 타이머를 켜고 집중 시간을 기록합니다.
  3. 과제 완료 처리: 완료한 과제를 체크해 XP를 얻습니다.
  4. 성장 확인: 대시보드에서 XP, 레벨, 완료율, 집중 시간을 확인합니다.

구현 기술

사용한 기술

  • HTML5 semantic tag 기반 페이지 구조
  • CSS3 layout, grid, flexbox, media query
  • JavaScript DOM 조작, event, array, object, function
  • localStorage 기반 과제, 집중 기록, 설정 저장
  • setInterval 기반 집중 타이머

성장 시스템

XP 계산 방식

  • 긴급 과제 완료 시 35 XP를 얻습니다.
  • 보통 과제 완료 시 25 XP, 여유 과제 완료 시 15 XP를 얻습니다.
  • 집중 시간은 누적 25분마다 10 XP로 계산됩니다.
  • 100 XP를 모을 때마다 다음 레벨로 올라갑니다.
  • XP는 저장된 과제와 집중 기록을 기준으로 자동 계산됩니다.

데모 데이터

프로젝트 데이터 초기화

Campus Quest의 localStorage 데이터를 지우고 예시 과제 4개를 다시 생성합니다.

큐비 성장 기록

레벨별 진화 단계

현재 저장된 데이터 기준 레벨을 확인합니다.

레벨 1

큐비

파란 젤리 몸과 작은 귀를 가진 기본 단계입니다.

레벨 3

스타 큐비

민트빛으로 변하고 작은 반짝이가 생깁니다.

레벨 5

윙 큐비

보라색으로 성장하고 집중 날개가 열립니다.

레벨 8

아스트라 큐비

금빛 최종 단계로 왕관과 날개를 함께 갖습니다.

진화 확인

레벨별 진화 빠르게 확인하기

버튼을 누르면 예시 과제로 초기화한 뒤, 선택한 레벨에 맞는 집중 기록을 자동으로 추가해 진화 단계를 바로 확인합니다.
