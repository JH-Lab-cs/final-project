Campus Quest

1. 프로젝트명
Campus Quest

2. 주제
대학생 과제와 공부 시간을 관리하는 게이미피케이션 웹사이트입니다. 과제를 퀘스트처럼 등록하고, 완료하면 XP를 얻으며, 집중 타이머 기록을 통해 레벨을 올릴 수 있습니다.

3. 실행 방법
압축을 푼 뒤 index.html 파일을 브라우저에서 직접 엽니다. 서버, 데이터베이스, npm, CDN, 외부 이미지, 외부 폰트, 외부 API는 필요하지 않습니다.

4. 주요 페이지
- index.html: 레벨, XP, 완료율, 누적 집중 시간, 남은 과제, 마감 임박 과제를 보여주는 대시보드입니다.
- tasks.html: 과제 추가, 입력 검증, 완료/완료 취소, 삭제, 필터, 검색, 정렬을 제공하는 과제 관리 페이지입니다.
- focus.html: 시작, 일시정지, 초기화, 세션 저장, 기록 삭제, 전체 삭제, 집중 통계를 제공하는 집중 타이머 페이지입니다.
- guide.html: 사용 방법, 구현 기술, XP 계산 방식, 캐릭터 진화 단계, 데모 데이터 초기화 기능을 보여주는 안내 페이지입니다.

5. 주요 기능
- localStorage를 사용해 과제, 집중 세션, 설정을 저장합니다.
- 첫 접속 시 예시 과제 4개를 자동 생성합니다.
- 마감일을 기준으로 D-Day, D-숫자, D+숫자, 완료 상태를 표시합니다.
- 완료되지 않았고 2일 이내 마감이거나 지난 과제를 마감 임박 과제로 표시합니다.
- 완료 과제와 누적 집중 시간을 기반으로 XP와 레벨을 계산합니다.
- XP는 저장하지 않고 매번 계산하여 중복 지급을 방지합니다.
- 대시보드에서 레벨별 캐릭터 진화 경로를 보여줍니다.
- 과제 카드에서 중요도별 획득 가능 XP 보상을 바로 확인할 수 있습니다.
- 집중 타이머 페이지에서 1분 기록 준비 기능으로 저장 조건을 바로 확인할 수 있습니다.
- 1분 기록은 저장 기능 확인용이며, 집중 XP는 누적 25분 이상부터 증가합니다.
- 가이드 페이지의 레벨 확인 버튼으로 발표 시간 안에 캐릭터 진화 단계를 빠르게 확인할 수 있습니다.
- 다크모드 설정을 저장합니다.
- 모바일 화면에서 메뉴와 카드 레이아웃이 반응형으로 바뀝니다.

6. 사용 기술
- HTML5 semantic tag
- CSS3 layout, grid, flexbox, media query, CSS variable
- JavaScript DOM 조작
- JavaScript event 처리
- JavaScript array, object, function, condition, loop
- localStorage
- setInterval

7. 시연 순서
1. index.html에 접속합니다.
2. 대시보드에서 레벨, XP, 통계를 확인합니다.
3. tasks.html로 이동합니다.
4. 새 과제를 추가합니다.
5. 필터, 검색, 정렬을 확인합니다.
6. 과제를 완료 처리합니다.
7. XP와 완료율 변화를 확인합니다.
8. focus.html로 이동합니다.
9. 타이머 시작, 일시정지, 1분 기록 준비, 저장을 확인합니다.
10. 집중 기록과 누적 집중 시간을 확인합니다.
11. 새로고침 후 데이터가 유지되는지 확인합니다.
12. 화면 크기를 줄여 반응형 레이아웃을 확인합니다.

8. 수업 요구사항 충족 여부
- 웹페이지 4개 구성
- 여러 이벤트 처리 기능 구현
- 반응형 웹 지원
- HTML, CSS, JavaScript 파일 분리
- localStorage 사용
- 서버 없이 index.html 직접 실행 가능
